# pcb-design-using-Altium-Designer

PCB project includes schematic design, layout, and fabrication-ready Gerber files.

## Project 1: IR Sensor
### Overview
This project involves designing a PCB for an IR sensor. The design includes the schematic and layout necessary for the sensor to function effectively.

### Key Files
- **IR sensor pcb.OutJob**: Output job file for the PCB.
- **IR sensor pcb.PrjPcb**: Project file for the PCB design.
- **PCB2.PcbDoc**: PCB document containing the layout.
- **Sheet2.SchDoc**: Schematic document for the IR sensor.

### Steps
1. Open the project in Altium Designer.
2. Review the schematic in `Sheet2.SchDoc`.
3. Inspect the layout in `PCB2.PcbDoc`.
4. Generate Gerber files for fabrication.
